# Dragon – ANTI-RE PYTHON

##  What it does
Anti-reverse-engineering builder for Python projects.

It wraps:
- PyArmor 6.8.1
- Nuitka build system
- UPX compression
- Anti-debug techniques

##  How to use
1. Place `Dragon.exe` in your project folder  
2. Run it  
3. Input your project folder name and main file  
4. It builds a hardened exe output

 Some antivirus might falsely flag outputs as malware — this is common with heavy protection tools.
